# Compare CSV & Excel Files Online

This is a simple online tool to compare two CSV files or Excel spreadsheets and instantly see what changed.

You can:
- Upload two CSV or Excel files
- Identify added rows, removed rows, and changed values
- Download the comparison results

No account required. One free comparison is available.
